# Restaurant-SQL-Data-Exploration-Project
📋 Project Description

This project is a basic data exploration using SQL to analyze two main tables: menu_items and order_details. The goal is to gain insights into the restaurant's menu and customer ordering behavior.

🧾 Tables Used

menu_items: Contains information about each menu item (name, category, and price).

order_details: Contains information about each item ordered, including the order ID and date.
